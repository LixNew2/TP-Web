----------- Après avoir décompressé le fichier "TP_SITE_WEB_BOTTON_GAEL.7z" -----------

RECOMMANDATION : Utilisez VSCode pour être sûr de le faire fonctionner correctement !

Installer VSCode : https://code.visualstudio.com/

Une fois VSCode installé. Lancez-le et rendez-vous dans les extensions sur le côté gauche.
Dans les extensions, tapez "PHP Server" et téléchargez celui avec une icône bleue.

Une fois l'extension téléchargée, téléchargez PHP (NTS) : https://windows.php.net/download

Après avoir téléchargé PHP, allez dans VSCode, puis dans les extensions installées et cliquez sur l'engrenage en bas à droite de l'extension "PHP Server", ensuite cliquez sur "Extension Settings". Dans les paramètres de l'extension, collez le chemin d'accès du fichier "PHP.exe" dans la catégorie "Phpserver : PHP Path".

Une fois que toutes ces étapes ont été correctement effectuées, faites un clic droit sur le dossier "TP_SITE_WEB_BOTTON_GAEL", puis ouvrez avec VSCode. Une fois ouvert dans VSCode, déroulez le dossier "files", puis le dossier "SITE". Dans site, cliquez sur le fichier "index.php", puis sur l'icône bleue en haut à droite de VSCode pour lancer le serveur PHP avec ce fichier.

PS : Dans "files", vous retrouverez un dossier "SITE" qui contient le site web, et un dossier par TP (TP1, TP2, etc...) avec dans chaque dossier un dossier par exercice (EX1, EX2, etc..).
